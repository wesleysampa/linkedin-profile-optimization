# Contribuindo com o Projeto

Obrigado por considerar contribuir!

## Como contribuir
- Fork o repositório
- Crie uma branch descritiva
- Faça commits claros
- Envie um Pull Request explicando a mudança

## O que pode ser contribuído
- Melhorias na documentação
- Novos exemplos
- Ajustes nos prompts
- Traduções

## Padrões
- Não alterar a arquitetura principal
- Manter clareza e rastreabilidade
- Evitar simplificações que prejudiquem senioridade
