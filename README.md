## 👋 Comece por aqui

Siga exatamente essa ordem:

```text
1. Medir seu nível atual (SSI)
2. Ajustar configurações básicas
3. Coletar dados do seu perfil
4. Definir seu posicionamento (PROFESSIONAL_CONTEXT)
5. Executar os prompts
6. Aplicar melhorias
7. Acompanhar evolução
```

⏱️ Tempo médio: **60 a 90 minutos**

---

## 🎯 O que você vai conseguir

* Ser encontrado por recrutadores técnicos
* Melhorar compatibilidade com ATS
* Corrigir seu posicionamento (Pleno / Sênior / Especialista)
* Ter um perfil claro, estratégico e com impacto
